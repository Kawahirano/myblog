搭建自己的导航页

Kawahirano2024-03-122024-03-12

1️⃣ 项目简介

项目地址： https://github.com/xjh22222228/nav

这款导航站和其他导航站最大的区别就是——这个导航站内置了八百多个实用网站，部署好了咱们可以直接用，非常方便

2️⃣ fork 代码到自己的仓库

先把项目 fork 到自己的仓库

以后作者如果有更新，兄弟们直接点击 Sync fork 就可以同步更新了

3️⃣ Vercel 部署 nav

打开 https://vercel.com/

不得不说 Vercel 真的是一个好东西，以后二冰也会更新很多用 Vercel 部署的项目，绝对的生产力工具

找到刚刚 fork 的项目 >> Import

import

可以修改 Vercel 项目名称 >> Deploy

deploy

搭建完成，是的，就是这么简单。

4️⃣ 设置自定义域名

vercel 分配的域名，因为众所周知的原因，我们无法直接访问，这里我们配置一下自己的域名，二冰只有一个阿里云的域名 twoice.fun ，以下以此域名为例

  • 添加记录
  • 记录类型：CNAME-将域名指向另外一个域名
  • 主机记录：以 mind 为例
  • 记录值：cname.vercel-dns.com
  • 确认即可

解析一个子域名

这样我们就解析了一个子域名 mind.twoice.fun

返回刚刚创建的 Vercel 项目

  • Settings >> Domains
  • 输入我们刚刚解析的子域名 mind.twoice.fun
  • add

Vercel 添加子域名

等个一两分钟就添加好了

5️⃣ enjoy it

输入你自己的子域名 enjoy it

二冰这里输入的是 mind.twoice.fun

默认主题：light

light

除了默认主题之外，还有另外的四种主题，兄弟们可以选择自己喜欢的主题

Sim

Side

Shortcut

App

6️⃣ 导航站后台

是的，导航站有后台，你可以登录后台增删改减你的导航站

①、申请 github 的 token

通过这个链接申请一个 token，如果不会操作，自行百度申请一下吧

https://github.com/settings/tokens

①、在你刚刚 fork 的仓库 >> branch

输入 image >> Create New branch

创建一个

③、修改项目配置文件 nav.config.ts

点击仓库的 nav.config.ts 文件 >> 点击修改 >> 将地址换成自己的仓库地址

修改项目配置文件 nav.config.ts

修改完别忘了点一下 Commit changes...

Commit changes…

等一会儿输入 你自己的域名/#/system 即可访问

这里我们输入 mind.twoice.fun/#/system

输入刚刚申请的 token

你的域名/#/system

就可以进入后台啦

成功进入后台

好了，今天的分享就到这了，更多的功能兄弟们可以自行探索，真的是一款良心的导航站。

有需要的兄弟们，赶紧操练起来吧！如果你有什么 想玩的项目 或者 想要解决什么问题 都可以 留言 或者 私信 跟二冰说，二冰肯定给你优先安排
